Sposoby podłączania CSS do kodu strony WWW

1. Zewnętrzny arkusz stylów

Arkusz stylów definiowany w osobnym pliku CSS i linkowany do pliku HTML.

      <HEAD>
        <LINK rel="stylesheet" type="text/css" href="style.css">
      </HEAD>

2. Styl lokalny (wpisany, liniowy)

Arkusz stylów definiowany bezpośrednio w sekcji head dokumentu HTML.

      <HEAD>
        <STYLE type="text/css">
          body {
            font-family: Arial, sans-serif;
            font-size: 16px;
          }
          h1 {
            color: #333;
            text-align: center;
          }
        </STYLE>
      </HEAD>

3. Wewnętrzny arkusz stylów

Arkusz stylów definiowany w elemencie <style> w sekcji body dokumentu HTML.

      <body>
        <style>
          body {
            font-family: Arial, sans-serif;
            font-size: 16px;
          }
          h1 {
            color: #333;
            text-align: center;
          }
        </style>
        ...
      </body>

4. Alternatywny arkusz stylów

Możliwość definiowania kilku arkuszy stylów i wyboru domyślnego lub w zależności od preferencji użytkownika.

      <HEAD>
        <LINK rel="stylesheet" type="text/css" href="style.css" title="Domyślny">
        <LINK rel="alternate stylesheet" type="text/css" href="style1.css" title="Styl 1">
        <LINK rel="alternate stylesheet" type="text/css" href="style2.css" title="Styl 2">
      </HEAD>

5. Import arkusza stylów

Możliwość włączania arkusza stylów zdefiniowanego w innym pliku za pomocą dyrektywy @import.

      <HEAD>
        <STYLE type="text/css">
          @import url("zewnetrzny.css");
        </STYLE>
      </HEAD>
